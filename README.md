# Tech Layoffs Analysis — Q1 2026

## Overview
Analysis of 101K+ jobs cut across 27 tech companies between 
Jan–Mar 2026, combining Python NLP and Power BI.

## Key Findings
- 101K jobs cut in Q1 2026 — nearly 2x the full-year 2025 total
- AI restructuring drove the most layoff events (13/28)
- Cost-cutting rounds cut deeper on average (12.9% of workforce)
- 54% of companies created AI roles alongside layoffs

## Tools & Methods
- Python (NLP) — classified unstructured company statements 
  into 5 reason categories using keyword extraction
- Power BI — 6-panel dashboard with custom DAX measures
